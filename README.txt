Views Database Connector

i. Installation
Upload this module's folder to your server and place it in the
sites/all/modules folder.  After the upload is finished, enable the module.

ii. Configuration
Add extra databases following the instructions found in the installation's
sites/default/settings.php file.

iii. Utilization
When you add a new view, you should now be able to pick a new entry in the
"Show" select menu.  Each item created by this module will be prefixed by
"[VDC]"
